# Prostokat

Obliczanie pola i obwodu prostokata

## Lista krokow

1. Pobierz od uzytkownika bok a
2. Pobierz od uzytkownika bok b
3. Oblicz obwod prostokata
4. Oblicz pole prostokata
5. Wyswietl wynik obwodu
6. Wyswietl wynik pola

## Pseudokod

START
input a
input b
pole = a\*b
obwod = (2\*a) + (2\*b)
output pole
output obwod
STOP
